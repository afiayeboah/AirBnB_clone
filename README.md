## How to Start It ##
First, Clone the project repository: https://github.com/afiayeboah/AirBnB_clone.git
Then, go to the project directory:cd AirBnB_clone
Install dependencies (if any):pip install -r requirements.txt
Now, Run the command interpreter:./console.py

## How to Use It ##
Once the command interpreter is running, you can enter commands to interact with the AirBnB clone.

## Examples ##
To show all available commands, run : (hbnb) help
To create a new instance run: (hbnb) create BaseModel
